# GISVENTURE Website

This is the personal project and profile site for Srijith Balan, showcasing open-source geospatial tools and initiatives like CartoZen.

## 🚀 How to Deploy on GitHub Pages

1. Create a repository named `gisventure`
2. Upload all files from this folder (not the folder itself)
3. Go to your repo > Settings > Pages
4. Under Source, choose: `main` branch and `/ (root)`
5. Save — your site will be live at:
   https://yourusername.github.io/gisventure/

## 🌐 Custom Domain (Optional)
If you own `gisventure.com`, add a `CNAME` file with the domain inside, and configure it in GitHub Pages settings.

## 📄 Included
- Home (`index.md`)
- About (`about.md`)
- Tools (`tools.md`)
- `_config.yml`
- `assets/` folder for logos/images
